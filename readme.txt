== Description ==

= FREE WIDGETS =
PickleJar Live for Artists & Venues includes a variety of widgets for you to build professional and engaging websites:

- **PickleJar Event List**. Add eye-catching headlines.
- **PickleJar Artist List**. Control the size, opacity and other settings of images.

== Installation ==

= Minimum Requirements =

* WordPress 6.1.1 or greater
* PHP version 7.0 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 6.1 or greater
* WordPress Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to PJ Integration and follow the login process
4. Select the page to show the Artists List and another page to show the Events List.
5. You can create multiple configuration using PJ Events or PJ Artist menu items and use them into any page.

== Frequently Asked Questions ==

**How do I install PickleJar Live for Artists & Venues?**

To install PickleJar Live for Artists & Venues, follow the steps below:
From your WordPress dashboard -> Go to Plugins -> Click on 'Add new'-> In the Search field, enter PickleJar Live for Artists & Venues and choose PickleJar Live for Artists & Venues.
Press install -> After installation, click Activate.

**Does PickleJar Live for Artists & Venues work with all the themes?**

PickleJar Live for Artists & Venues works all the themes that respect the coding standards of WordPress set by its Codex.

**Is PickleJar Live for Artists & Venues compatible with Gutenberg?**

PickleJar Live for Artists & Venues and Gutenberg work seamlessly together.

**Does it work with other WordPress plugins? **

It works with almost all the plugins. If you experience an incompatibility issue, please report it to us and to the plugin that conflicts with PickleJar Live for Artists & Venues.

**Do I need to know how to code?**

No! PickleJar Live for Artists & Venues provides you with all the widgets and features that you need to include  website without using code.
